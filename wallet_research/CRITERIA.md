# Polymarket Bot Wallet Research — Screening Criteria

**Last updated**: 2026-04-20 (v2 — mid-tier focus)
**Purpose**: 수익성 입증된 Polymarket 봇 지갑을 수집·분석해서 전략 카탈로그 구축.
카피트레이딩이 아닌 **전략 카테고리 학습**이 목적.

---

## Tier 1 — 최상위 고래 ($1M+ 월 PnL) [수집 완료, 참고용]

- 대상: 월간 리더보드 Top 20
- 스크리닝: 거래 수 ≥ 1,000 / 일평균 ≥ 20 / 단일 카테고리 ≥ 60% 중 2개
- 결과: 7개 봇 수집 (`wallets/` 참조)
- **한계**: 자본·속도가 엣지의 상당 부분 → 네 $500~$3K 자본으로 복제 난이도 높음
- **용도**: 전략 카테고리 범위 파악, 카피 X

---

## Tier 2 — 중위권 ($5K~$500K 월 PnL) [현재 수집 대상]

### 봇 판별 (3개 중 2개 이상)
| # | 지표 | 컷오프 |
|---|---|---|
| 1 | 누적 거래 수 | ≥ 1,000건 |
| 2 | 최근 30일 일평균 | ≥ 20건/일 |
| 3 | 단일 카테고리 집중도 | ≥ 60% |

### 수익·스케일 (전부 통과해야 함)
- 월 PnL **$5,000 ~ $500,000**
- 최근 30일 PnL **양수**
- **최근 3개월 중 2개월 이상 양수 PnL** (지속성 — 운빨 배제)
- Biggest win ≤ **$100,000** (한 방 대박 디스크리셔너리 배제)
- 현재 포지션 ≤ **$200,000** (자본 체급 매칭)

### 자동 제외
- 월 PnL 음수 or 누적 PnL 음수
- 거래 수 < 1,000
- Biggest win > $100K
- 현재 포지션 > $200K
- 최근 3개월 중 2개월 이상 음수 (지속성 실패)
- Tier 1 주소와 중복

### 다양성 (소프트 가이드, 강제 X)
- 크립토 / 이벤트 / esports 최소 1개씩 포함되면 좋음
- 샘플 확보가 우선

### 지속성 데이터 미확인 시
- 프로필 PnL 차트 시각 판단
- 거래 history 월별 빈도 참고
- 확인 불가 → 카드에 "지속성 미검증" 노트, 수집은 진행

---

## 카드 필수 필드

### 본체
- **전략 본체** (어떻게 돈 버는가)
- **실행 요건** (자본 / 속도 / 인프라 / 데이터 / 도메인)
- **복제 난이도**

### 부가 기록용 (배제 필터 아님)
- PnL / Volume 비율 (엣지 질 참고; <1% = MM, 1~15% = 정상, >30% = 의심)
- 신뢰도 노트 (데이터 결손 지점 명시)

## 향후 확장 트리거
- 자본 $10K 돌파 시: Tier 3 (월 PnL <$5K 마이크로 봇) 추가 수집
- Dune Analytics API 접근 확보 시: 수동 스크레이프 → 자동화 전환
- 주간 리더보드 교차 분석으로 단기 vs 장기 봇 구분
